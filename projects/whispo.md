# Whispo — SaaS голосовой диктовки

Коммерческий продукт для голосового ввода текста с AI-обработкой: нативное
macOS-приложение, FastAPI-бэкенд и веб-портал. Полный цикл — от записи звука по
горячей клавише до транскрибации, форматирования и вставки в активное приложение,
плюс подписки, платежи и наблюдаемость.

## Компоненты

- **macOS-приложение** на Swift/SwiftUI (~12,5 тыс. строк, 40+ файлов) — запись по
  глобальным хоткеям, транскрибация через Groq Whisper, вставка в активное приложение
  через буфер обмена + эмуляцию ⌘V (Accessibility API), Memory Storage с синхронизацией
  в облако, история, настройки, оверлей с анимацией волны
  - **Три режима ввода:** правый Option — диктовка с мгновенной вставкой (push-to-talk);
    Option+Space — сохранение в Memory Storage; правый Shift — toggle-запись; Escape — отмена
- **Backend** на FastAPI — 15 модулей API (роутеров): транскрибация, подписки,
  платежи, Memory Storage, рефералы, AI-обработка, метрики, cron, аккаунт
- **Web-портал** на React 19 + TypeScript + Vite — лендинг и личный кабинет:
  аутентификация (Supabase), управление подпиской и картой, реферальная программа,
  безопасность аккаунта, скачивание, FAQ, юридические страницы; билингва EN/RU

## Аутентификация и безопасность

- **Supabase JWT** с поддержкой двух алгоритмов: ES256 (новые токены, проверка через
  JWKS) и HS256 (legacy)
- **Rate limiting** на `slowapi` + Redis: лимиты по пользователю (из JWT `sub`) и по IP
  (реальный IP из `X-Real-IP` за nginx); строгие лимиты на транскрибацию (30/мин),
  AI (20/мин) и платёжные вебхуки (10/мин)
- **Защита от prompt injection** в пользовательских AI-промтах: нормализация Unicode,
  вырезание zero-width/bidi-символов, фолдинг гомоглифов (кириллица/греческий → латиница),
  удаление role-switch токенов, песочница `<user_formatting_preferences>`

## Монетизация

- Тарифы: **Free / Standard** (+ Custom для enterprise через форму)
  - Free: 1000 слов/день, 1 хранилище, модель `whisper-large-v3-turbo`
  - Standard: 10 часов транскрибации/мес, `whisper-large-v3`, AI-обработка, безлимит хранилищ
- Два платёжных шлюза: **YooKassa** (₽299/мес) и **CryptoCloud** (крипта, $4.99/мес),
  годовая подписка со скидкой
- Автопродление по сохранённой карте, идемпотентная обработка вебхуков, логирование транзакций
- **Реферальная программа:** бонусные месяцы за приглашённых друзей (порог 3), trial по рефералу

## AI-функции

- **Транскрибация:** Groq Whisper (`whisper-large-v3` / `whisper-large-v3-turbo` по тарифу),
  retry с экспоненциальным backoff
- **Реструктуризация текста:** Groq LLM `llama-3.3-70b-versatile` — чистит и
  структурирует «сырой» голосовой ввод (дневной лимит, защита от abuse)
- Мультиязычная транскрибация

## Инфраструктура и наблюдаемость

- **Supabase** (PostgreSQL + Auth + RLS), доступ из бэкенда через PostgREST с
  атомарными RPC-счётчиками; 12 SQL-миграций
- **Redis** — кэш (подписки, хранилища) и хранилище счётчиков rate limit
- **Docker Compose** (7 сервисов), **nginx** + **Cloudflare**, systemd-cron (продление/
  истечение/уведомления)
- Полный стек мониторинга: **Prometheus + Grafana** (3 бизнес-дашборда) + **Loki/Promtail**
  + node-exporter, алертинг в Telegram
